  ## AI-assisted engineering

  I use **Claude Code** as a primary development tool to move faster without lowering the engineering bar.

  - Explore unfamiliar codebases and trace behavior across modules
  - Turn requirements into small, reviewable implementation steps
  - Accelerate testing, refactoring, and technical documentation
  - Validate AI-assisted changes through diffs, automated checks, and runtime evidence

  AI supports the execution; architecture, trade-offs, and final responsibility remain human-owned.
